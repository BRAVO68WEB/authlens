# AuthLens

**AuthLens** is a comprehensive authentication playground and debugging tool for developers working with multiple authentication protocols including OIDC, OAuth 2.0, SAML, and standard API authentication.

> **Latest Update (v1.2.0)**: Added LoginRadius CIAM API integration with dynamic registration forms, SOTT token management, and auto-configuration from hosted pages! 🎉

## Features

### Multi-Protocol Support
- **OpenID Connect (OIDC)**: Authorization Code, Implicit, and Hybrid flows with PKCE support
- **OAuth 2.0**: All standard grant types including device code flow
- **SAML 2.0**: SP-initiated and IdP-initiated flows with complete ACS callback support ✨
- **LoginRadius CIAM API**: Registration, Login, User Info, and Site Config endpoints with dynamic form schema ✨
- **API Testing**: Custom HTTP requests with headers and body

### Developer Tools
- **JWT Validator**: Decode and verify JSON Web Tokens with JWKS support
- **Claim Checker**: Validate JWT claims with custom rules
- **Request Builder**: Build and test API requests
- **JWK Viewer**: Inspect JSON Web Key Sets
- **XML Formatter**: Format and decode SAML XML responses

### Key Capabilities
- 🔍 **Auto-Discovery**: Automatic OIDC configuration via well-known endpoints
- 🔐 **PKCE Support**: Secure authorization code flows for public clients
- 📝 **Comprehensive Logging**: Detailed request/response logs with redaction
- 💾 **Workspace Management**: Import/export provider configurations
- 🎨 **Modern UI**: Beautiful, responsive interface with dark mode support
- 🔧 **Developer-Friendly**: Built by developers, for developers
- 🚀 **LoginRadius Integration**: Auto-fetch configuration from hosted pages, dynamic registration forms, SOTT token management
